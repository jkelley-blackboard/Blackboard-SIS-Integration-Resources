# SIS Integration Framework Resources

Reference materials, field specifications, sample files, and custom scripting resources for Blackboard Learn SIS integrations.

These resources cover the two primary SIS integration types:

---

## [LIS (Learning Information Services)](lis/)

Resources for LIS-based SIS integrations, including custom field mapping scripts, XML samples, and the scripting API reference.

| Path | Contents |
|---|---|
| [`lis/scripting/`](lis/scripting/) | JavaScript field mapping scripts and integration guides |
| [`lis/xml-samples/`](lis/xml-samples/) | Sample LIS XML files for testing and reference |
| [`lis/data-models/`](lis/data-models/) | Data model reference for accessing SIS data in scripts |

---

## [Snapshot Flat File](snapshot/)

Resources for Snapshot Flat File integrations, including field-level specifications for every object type.

| Path | Contents |
|---|---|
| [`snapshot/specs/`](snapshot/specs/) | Field-level specs for all Snapshot object types (Markdown) |
| [`snapshot/operations/`](snapshot/operations/) | Guidance for running the integration day-to-day — constructing and sending requests, checking processing status, and automating recurring feeds |
